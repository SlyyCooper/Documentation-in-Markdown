# `chat_history`#

**Chat message history** stores a history of the message interactions in a
chat.

**Class hierarchy:**

    
    
    BaseChatMessageHistory --> <name>ChatMessageHistory  # Examples: FileChatMessageHistory, PostgresChatMessageHistory
    

**Main helpers:**

    
    
    AIMessage, HumanMessage, BaseMessage
    

**Classes**

`chat_history.BaseChatMessageHistory`() | Abstract base class for storing chat message history.  
---|---  
`chat_history.InMemoryChatMessageHistory` | In memory implementation of chat message history.
